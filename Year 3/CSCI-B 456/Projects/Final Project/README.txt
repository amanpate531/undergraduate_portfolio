Running instructions:

1. Find an image in the TestData folder you would like to test.

2. Edit variable "fileName" in line 2 to the corresponding file path.

3. Run

Note: If you want to view the output of MATLAB's OCR, you can either display the result 
using disp(ocrT) or you can display the guessed text using disp(ocr.Text) after ocr() is called.
